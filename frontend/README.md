### Como inicializar o sistema
  - É necessário ter instalado:
    * **node >= 8.10**;
    * **yarn**;
    * **npm >= 5.6** 
    

  - Clonar o projeto:
```sh
$ git clone git@github.com:Matheusvil/ProjetoENG3.git
``` 
  - Instalar as dependências:
  ```sh
$ yarn install ou npm install
``` 
  - Iniciar o servidor Web: 
  ```sh
$ yarn start ou npm start
```
